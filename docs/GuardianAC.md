# GuardianAC

GuardianAC is TurtleRec's anti-cheat and integrity system.

## Checks

- Speed hacking
- Fly hacking
- Teleport abuse
- Modified DLLs
- Packet tampering
- Memory editing
- Infinite ammo
- Invalid cosmetics
- Invalid room ownership
- Suspicious Photon events
- Abnormal movement authority
- Fake room instances
- Asset injection

## Actions

- Log detection
- Warn moderation team
- Pause affected instance
- Kick confirmed cheater
- Ban repeat offenders
